通过直接在工厂类上实现FactoryBean接口，实现了这个接口后，
不需要我们在配置文件（方法三）通过factory-bean factory-method factory-method 指定工厂类和工厂中实例化的方法
-----------------------------------------------------------------------------------------------------
具体步骤
    1创建具体产品对象
    2创建工厂对象，实现FactoryBean
    3在实现的方法中GetObject方法中new对象，（也可以在里面对对象进行加工）
    4配置类文件配置id class ，class写对象工厂的类路径 会自动返回你在get Object 中new 的对象
    5Test
